== Description ==
Tiny Wordpress plugin to display the web server host name & IP address in the Wordpress Menu Bar in green. Ideal if migrating servers or to quickly view which server a Wordpress installation is installed on.

== Installation ==
Simply activate the plugin. The host name & Ip address will be displayed in the top left on the Wordpress menu in green.
